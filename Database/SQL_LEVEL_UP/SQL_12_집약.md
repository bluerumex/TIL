### SQL 집약

>레코드 단위가 아닌 '집합' 단위로 처리하는 집합 지향(set-oriented) 사고로 코드를 작성해보자.
>SQL에는 집약 함수(aggregate function)라고 하는 집약함수가 있다
>COUNT, SUM, AVG, MAX, MIN
