### Java Stream

#### Stream
```{.java}
스트림이란?
물이 흐르는 개울.. IT용어로는 "순서가 있는 데이터의 연속적인 흐름"을 의미
데이터가 생성되는 소스(source)와 데이터를 소비하는 목적지(destination)를 가지고 있다.
스트림에서 데이터는 바이트도 될 수 있고, 문자, 객체도 될수 있다.

입출력 스트림의 종류
 * 바이트 스트림 (byte stream)
   - 8비트의 바이트 단위로 입출력하는 클래스들이며, 주로 이진 데이터를 읽고 쓰기 위해 사용된다.
     바이트 스트림의 클래스 이름에는 InputStream(입력)과 outputStream(출력)이 붙는다.
 * 문자 스트림 (character stream)
   - 문자 단위로 입출력하는 클래스이다. 자바는 기본적으로 유니코드를 사용하기 때문에 문자 스트림은 유니코드
     단위로 입출력한다. 문자 스트림 클래스 이름에는 Reader(입력)과 Writer(출력)이 붙는다. 

```
