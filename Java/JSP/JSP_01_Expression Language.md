### JSP 1. Expression Language_01

```
Expression Language란..
JSP 페이지에서 이용되는 여러 외부 데이터 객체를 쉽고 간편하게 참조하기 위한 언어
즉 표현언어는 <%= %>인 표현식 대신에 사용하거나 내장 객체 또는 액션태그에 저장된 저장된
자료를 쉽게 참조하기 위해 만들어진 언어이다.

JSP에서 브라우저 출력은 주로 표현식 태그를 이용..
<%= request.getParameter("userId") %>

위와 같은 태그를 표현식으로 표현하면,
${param.userId} 
${param['userId']}
${param["userId"]}

표현언어의 기본적인 문법
1. 표현언어는 $로 시작, 
2. 표현언어의 문장구조는 ${exp}와 같다.
3. 표현식 exp에서는 산술, 관계, 논리와 같은 기본적인 연산이 가능하다.
4. 표현식은 [객체명], [객체명.속성명], [객체명[첨자]], [객체명["속성명"]], [객체명['속성명']]과
   같은 구조로 구성된다.

표현언어의 자료 유형
정수형, 실수형, 문장열형, 논리형, null 5가지
문자열형은 'JSP', "JSP"와 같이 큰("), 작은(')따옴표 둘다 이용가능.

표현언어에서 이용되는 상수
논리값(Boolean), 자바에서 이용되는 정수형(1, -5), 실수형(3.1, 4.5E+4),
문자열은 'java', "java"둘다 가능, NULL


표현언어의 변수
표현언어에서 이용되는 변수는 pageContext.findAttribute(변수이름)으로 조회되는 변수의 평가값
으로 사용된다. 만일 다음과 같은 표현식이 있다고 가정할 경우

${product}

<<<<<<< HEAD
JSP 엔진은 page, request, sssion, application 범위에서 등록된 product를 평가하여 그 값을 반환한다.
만약 product를 찾을 수 없다면 null을 반환

물론 변수가 표현언어 내장 객체라면 그 내장 객체의 값을 반환
```
